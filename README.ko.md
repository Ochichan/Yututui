# ytm-tui

[English](README.md) · **한국어** · [日本語](README.ja.md)

*터미널 안에서 돌아가는 유튜브 뮤직 플레이어. 멀쩡한 버튼 달린 앱을 보고 "이걸 전부 키보드로 하고, 메모리도 덜 먹고, 카페에서 살짝 우쭐하게 만들면 어떨까?"라고 생각한 사람이 만들었습니다.*

`ytt`(앞으로 입력하게 될 명령어입니다)는 유튜브 뮤직 음악을 바로 터미널 창 안에서 재생합니다. 램을 조용히 갉아먹는 브라우저 탭도 없고, 썸네일 모양 광고도 없고, 마우스도 필요 없습니다 — 그리워질까 봐 마우스 지원은 넣어뒀지만요. 그냥 당신, 까만 화면 위의 글자들, 그리고 솔직히 좀 과하다 싶은 숫자의 키보드 단축키만 있으면 됩니다.

빠르고, 가볍고, 진작 잤어야 할 사람 한 명이 만들었습니다. 그 사람 컴퓨터에서는 잘 돌아갑니다. 이제 당신 컴퓨터에서도 그러길 바랍니다.

## 나한테 맞는 물건일까?

아마도요! 이런 분이라면 `ytm-tui`를 좋아할 가능성이 높습니다:

- 어차피 터미널에서 살고 있어서, 브라우저로 넘어가는 게 먼 산책처럼 느껴지는 분.
- 메모리도 거의 안 먹고 광고도 안 보이는 음악을 원하는 분.
- 키보드 단축키가 일종의 사랑의 언어라고 믿는 분.

반대로, 커다란 앨범 커버, 전부 마우스로 클릭하는 조작, 매끈한 폰 앱을 원한다면 **안 맞을** 겁니다. 괜찮아요. 그래도 우리 친구는 할 수 있습니다.

> **스크린샷에 대하여.** 없습니다. 음악 플레이어를 상상하시되, 전부 글자로 되어 있다고 생각하세요. 아니면 — 이게 더 낫겠네요 — 그냥 설치해서(2분이면 됩니다) 제 어설픈 캡처 실력 대신 진짜를 보세요.

## "그냥 음악. 지금 당장." (30초 버전)

맥을 쓰고 [Homebrew](https://brew.sh)가 이미 깔려 있다면:

```sh
brew install mpv yt-dlp                 # ytm-tui에 필요한 도우미 두 개
brew install Ochichan/tap/ytm-tui       # ytm-tui 본체
ytt                                      # 실행
```

그다음 `/`를 누르고, 노래 제목을 입력하고, `Enter`를 친 뒤, `↑`/`↓` + `Enter`로 하나 고르면 끝. 음악이 나옵니다.

그 외 모든 분(윈도우, 리눅스, "Homebrew가 뭐예요?")은 아래를 읽어주세요 — 그래도 쉽습니다, 약속해요.

## 1단계 — 도우미 두 개 설치하기

`ytm-tui`는 바퀴를 다시 발명하지 않습니다. 실제로 "소리를 찾아서 재생하는" 일은 사랑받는 무료 프로그램 두 개에게 조용히 맡깁니다:

- **[mpv](https://mpv.io)** — 스피커에서 소리가 나오게 해주는 녀석.
- **[yt-dlp](https://github.com/yt-dlp/yt-dlp)** — 유튜브에서 음원을 가져오는 녀석.

이 둘은 **딱 한 번만** 설치하면 됩니다. 운영체제를 고르세요:

| 내 컴퓨터 | 이렇게 입력 |
| --- | --- |
| **macOS** ([Homebrew](https://brew.sh) 사용) | `brew install mpv yt-dlp` |
| **Windows** ([Scoop](https://scoop.sh) 사용) | `scoop install mpv yt-dlp` |
| **Linux** (데비안/우분투) | `sudo apt install mpv yt-dlp` |
| **Linux** (아치) | `sudo pacman -S mpv yt-dlp` |

> **"패키지 매니저"가 뭔가요?** 터미널용 앱스토어 같은 겁니다. 한 줄로 프로그램을 설치해주죠. Homebrew(맥), Scoop(윈도우), apt/pacman(리눅스)이 대표적입니다. 표에 있는 게 아직 없다면, 링크를 눌러 먼저 설치하세요 — 이것도 한 번만 하면 됩니다.

제대로 됐는지 어떻게 아냐고요? `mpv --version`과 `yt-dlp --version`을 입력해 보세요. "command not found" 대신 글자들이 주르륵 나오면 성공입니다.

---

## 2단계 — ytm-tui 설치하기

**쉬운 방법** (추천) — 패키지 매니저에게 다 맡기기:

```sh
# macOS
brew install Ochichan/tap/ytm-tui

# Windows
scoop bucket add ytm-tui https://github.com/Ochichan/scoop-bucket
scoop install ytm-tui
```

**소스에서 직접 까는 방법** — 이 저장소를 폴더째로 받았다면:

```sh
# macOS / Linux
./install.sh

# Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

설치 스크립트는 예의가 바릅니다. 당신 컴퓨터에 맞는 완성품이 있으면 그걸 쓰고, 없으면 처음부터 직접 빌드합니다. 처음부터 빌드하려면 [Rust](https://rustup.rs)가 필요하고(먼저 설치하세요), 몇 분 걸리니, 커피 한 잔 내리러 가기 딱 좋은 순간입니다. 설치 위치를 `PATH`에 슬쩍 추가해서 어디서든 `ytt`를 실행할 수 있게 해주고, mpv와 yt-dlp가 있는지도 다시 확인해 줍니다.

## 3단계 — 실행하기

```sh
ytt
```

끝입니다. 세 글자.

가장 먼저 배워야 할 것: 언제든 **`?`**를 누르세요. 모든 키와 그 기능이 적힌, 항상 최신 상태인 전체 목록이 뜹니다 — 아래 내용을 다 잊어버려도, 앱이 대신 기억해 줍니다.

> **"`ytt: command not found`"이 뜬다면?** 터미널을 닫고 새 창을 연 뒤 다시 해보세요 — `PATH`가 따라잡을 시간이 필요합니다. 그래도 안 되면 [뭔가 잘못됐을 때](#뭔가-잘못됐을-때)를 보세요.

## 실제로 쓰는 법

`ytm-tui`는 여러 개의 **화면**으로 이루어져 있고, 그 사이를 오갑니다. 당신은 항상 그중 한 화면에 있습니다. 전체 지도는 이렇습니다:

- **🏠 플레이어** — 본진입니다. 지금 재생 중인 곡, 진행 막대, 볼륨, 그리고 미디어 버튼 한 줄이 보입니다. 거의 모든 게 여기서 시작됩니다.
- **🔍 검색** — `/`를 누르고, 원하는 걸 입력하고, `Enter`. `↑`/`↓`로 결과를 오가고, `Enter`로 선택한 곡을 재생합니다.
- **📚 라이브러리** — `l`을 누르세요. 즐겨찾기한 곡과 다운로드한 파일이 한곳에 모입니다. `All`, `Downloads`, 그리고 즐겨찾기를 둘러보세요.
- **📜 큐(재생목록)** — `c`를 누르세요. 다음에 나올 곡들의 줄입니다. `Enter`로 특정 곡으로 바로 점프하고, `Delete`로 줄에서 빼냅니다.
- **🎤 가사** — `L`(Shift+L)을 누르세요. 노래에 싱크된 가사가 있으면 음악에 맞춰 스크롤됩니다. 1인용 노래방이죠.
- **⚙️ 설정** — `,`(쉼표)를 누르세요. 모든 손잡이와 다이얼이 여기 있습니다. [내 입맛대로](#내-입맛대로-설정)를 보세요.
- **🤖 AI 비서** — `a`를 누르세요. 평범한 말로 대화하세요(선택 — 무료 키가 필요합니다. 아래 참고).

아무 패널이나 **닫고** 한 발 물러서려면 **`q`**. 앱 전체를 **종료**하려면 **`Ctrl+Q`**. 어디서든 플레이어로 순간이동하려면 **`Ctrl+H`**.

평범한 첫 사용은 이런 흐름입니다:

1. `/`를 눌러 노래 검색. → 2. `Enter`로 재생. → 3. 마음에 드는 곡에 `f`로 ♥ 즐겨찾기. → 4. `Ctrl+R`로 라디오를 켜서 음악이 영원히 이어지게. → 5. `,`로 설정에 들어가 예쁘게 꾸미기.

## 단축키

외우지 마세요. 앱에서 `?`를 누르면 치트시트가 바로 거기 있습니다. 그래도 궁금한 분을 위해, 평소에 자주 쓰는 것들입니다.

### 플레이어에서 (본진)

| 키 | 기능 |
| --- | --- |
| `Space` | 재생 / 일시정지 |
| `←` / `→` | 몇 초 뒤로 / 앞으로 |
| `↑` / `↓` | 볼륨 올리기 / 내리기 |
| `n` / `p` | 다음 / 이전 곡 |
| `f` | 현재 곡 평가 — 누를 때마다 🤔 중립 → 👍 좋아요 → 👎 싫어요 순환 (좋아요 = 즐겨찾기, 싫어요면 비슷한 곡이 줄어듭니다) |
| `/` | 검색 열기 |
| `l` | 라이브러리 열기 |
| `c` | 큐(다음 곡들) 열기 |
| `L` | 가사 보이기 / 숨기기 |
| `d` | 현재 곡 다운로드 |
| `s` | 셔플 켜기 / 끄기 |
| `r` | 반복: 끄기 → 전체 → 한 곡 |
| `a` | AI 비서 열기 |
| `,` | 설정 열기 |
| `?` | **모든** 키 보기 (앱 내 치트시트) |
| `Ctrl+R` | 라디오(자동재생) 켜기 / 끄기 |
| `Ctrl+H` | 플레이어로 돌아가기 |
| `Ctrl+Q` | 앱 전체 종료 |
| `q` | 뒤로 / 현재 패널 닫기 |

### 아무 목록에서나 (검색, 라이브러리, 큐)

| 키 | 기능 |
| --- | --- |
| `↑` / `↓` | 목록에서 위 / 아래 이동 |
| `Enter` | 선택한 항목 재생(또는 점프) |
| `f` | 즐겨찾기 |
| `d` | 다운로드 |
| `Delete` | 제거(큐나 라이브러리에서) |
| `q` | 패널 닫기 |

### 소리에 진심인 분을 위해

| 키 | 기능 |
| --- | --- |
| `e` | 이퀄라이저 프리셋 순환 |
| `N` | 음량 노멀라이즈 켜기/끄기 (조용한/시끄러운 곡을 비슷하게) |
| `>` / `<` | 재생 속도 빠르게 / 느리게 |

> **한글 키보드 팁:** 영어 입력으로 안 바꿔도 됩니다 — 단축키가 두벌식 자모를 알아듣거든요. `ㅂ`이 `q`처럼 동작합니다. 사소하지만 기분 좋은 부분이죠.

위의 모든 키는 **재설정 가능**합니다 — "뒤로"가 `q`인 게 거슬린다면, 바꾸세요(설정 → 단축키).

## 내 입맛대로 (설정)

`,`를 눌러 설정을 엽니다. 위쪽 탭은 `Tab` / `Shift+Tab`으로 옮기고, `↑`/`↓`로 위아래로 움직이고, `←`/`→`로 값을 바꿉니다(텍스트 칸은 `Enter`). 다 했으면 `q`를 누르세요 — 나가는 길에 **자동 저장**됩니다.

각 탭이 하는 일:

- **일반** — 쿠키 파일 위치, 다운로드 저장 위치, 마우스 지원 켜기/끄기, 앨범 아트 켜기/끄기, 실행 시 자동 재생 여부, 그리고 "초기화" 버튼 두 개(하나는 키만, 하나는 *전부* — 일을 너무 벌였을 때를 위해).
- **재생** — 재생 속도, `←`/`→` 탐색이 얼마나 점프할지, 그리고 *갭리스* 재생(곡 사이의 짧은 정적을 없앱니다).
- **EQ** — 미리 만들어진 프리셋(Flat, Bass Boost 등)이 있는 10밴드 이퀄라이저, 그리고 속삭이듯 조용한 곡과 벽을 때리는 곡이 비슷한 음량으로 나오게 하는 **Normalize** 토글.
- **AI** — 비서의 모델 선택, 키 붙여넣기, 자동재생 **라디오** 켜기, 라디오가 얼마나 모험적일지 설정.
- **테마** — 색 테마 고르기. 여러 개 있습니다. 그중 하나는 당신의 영혼과 맞을 거예요.
- **색상** — 프리셋이 영 별로다? 16진수 코드(`#ff8800` 같은)로 색을 하나하나 직접 고르세요.
- **단축키** — 어떤 단축키든 원하는 키로 재설정.

> **작은 함정:** 일부 설정에는 **"(재시작 후 적용)"**이라고 적혀 있습니다 — 마우스, 앨범 아트, 갭리스 같은 것들이요. 이건 `ytt`를 껐다 켜야 적용됩니다. 정상입니다. 앱이 당신을 무시하는 게 아니에요.

## 라디오: 멈추지 않는 음악

**`Ctrl+R`**를 누르면 `ytm-tui`는 지금 듣고 있는 곡을 중심으로 한 작은 라디오 방송국이 됩니다. 큐가 바닥나기 시작하면, 분위기에 어울리는 곡을 조용히 줄 세워 둡니다. 그래서 음악이 그냥... 멈추질 않아요. 게으른 사람의 꿈이죠. 이건 칭찬입니다.

얼마나 모험적일지는 **설정 → AI → 라디오 모드**에서 고를 수 있습니다:

- **집중** — 집 근처에 머뭅니다. 지금 듣는 것과 더 비슷한 곡들.
- **균형** — 합리적인 중간. 무난한 기본값.
- **발견** — 길을 벗어나 아직 못 들어본 것들을 찾아 떠납니다.

다음 곡을 *어떻게* 고르는지는 철저히 지켜지는 영업 비밀입니다 — 솔직한 대답이 "수많은 시행착오와 약간의 조용한 자부심"이기 때문이죠. 그냥 잘 됩니다. 즐기세요.

## AI 비서 (선택)

이건 보너스이고, 완전히 선택 사항입니다. 무료 **Google Gemini API 키**가 있다면, `a`를 누르고 음악 플레이어에게 평범한 말로 *말을 걸* 수 있습니다:

- "로파이 좀 틀어줘"
- "청소할 때 들을 신나는 곡 세 개 큐에 넣어줘"
- "지금 뭐 나오고 있어?"
- "이거 건너뛰고 더 차분한 거 틀어줘"

무슨 뜻인지 알아듣고, 음악을 찾아서, 대신 해줍니다. 명령어를 외울 필요가 없어요.

켜는 법: 구글 AI Studio에서 키를 받고(무료입니다), **설정 → AI → API 키**에 붙여넣으면 됩니다. 설정은 그게 전부입니다.

키가 없어도 문제없습니다 — 비서가 그냥 잠들어 있을 뿐, 앱의 다른 모든 기능은 그것 없이도 완벽하게 돌아갑니다.

## 다운로드는 어디에 저장되나

곡 위에서 `d`를 누르면 디스크에 저장되어 오프라인에서도 들을 수 있습니다. 기본 저장 위치는 여기입니다:

- **macOS:** `~/Music/ytm-tui`
- **Windows:** `%USERPROFILE%\Music\ytm-tui`
- **Linux:** 음악 폴더 안의 `ytm-tui` 하위 폴더

파일은 그 폴더 하나에 바로 들어갑니다 — 아티스트별 하위 폴더를 뒤질 필요 없습니다. 다운로드한 곡은 라이브러리의 **All**과 **Downloads**에 자동으로 나타나, 스트리밍 음악 옆에 사이좋게 자리 잡습니다.

다른 곳에 두고 싶다면? 설정의 **다운로드 폴더**를 바꾸거나, `YTM_DOWNLOAD_DIR` 환경 변수를 설정하세요.

## 쿠키로 로그인하기 (선택)

짧게: **아마 필요 없습니다.** 로그인하지 않아도 `ytm-tui`는 공개된 곡을 익명으로 잘 검색하고 재생합니다.

로그인이 필요한 경우는, 로그인된 유튜브 뮤직 계정이 있어야 들을 수 있는 곡(멤버십 전용, 지역 제한 곡 등)에 접근할 때뿐입니다. 그러려면 앱에 브라우저의 "쿠키"를 줍니다 — 당신이 당신임을 증명하는 작은 파일이죠.

기본적으로 `cookies.txt` 파일을 여기서 찾습니다:

- **macOS:** `~/Music/ytm-tui/cookies.txt`
- **Windows:** `%USERPROFILE%\Music\ytm-tui\cookies.txt`

그 폴더를 만들고, 브라우저에서 유튜브 / 유튜브 뮤직 쿠키를 **Netscape 형식**의 `cookies.txt`로 내보낸 뒤(브라우저 확장 프로그램으로 할 수 있습니다), 그 파일을 넣고, `ytt`를 다시 시작하세요.

```sh
# macOS — 폴더 만들기
mkdir -p ~/Music/ytm-tui
```

```powershell
# Windows (PowerShell) — 폴더 만들기
New-Item -ItemType Directory "$HOME\Music\ytm-tui" -Force
```

**설정 → 쿠키 파일**에서 다른 파일을 가리킬 수도 있습니다.

> ⚠️ **이 파일은 비밀번호처럼 다루세요.** 로그인된 브라우저 세션처럼 동작할 수 있으니, 공유하지 말고, 저장소에 커밋하지 말고, 어디에도 올리지 마세요.

## 뭔가 잘못됐을 때

심호흡하세요. 거의 항상 이 중 하나입니다:

| 증상 | 해결 |
| --- | --- |
| **아무것도 안 나오거나, 재생하자마자 에러가 난다** | mpv나 yt-dlp가 설치 안 됐거나 `PATH`에 없을 겁니다. [1단계](#1단계--도우미-두-개-설치하기)를 다시 하세요. |
| **`ytt: command not found`** | 완전히 새 터미널 창을 여세요. 그래도 그러면 설치 폴더가 `PATH`에 없는 겁니다 — 설치 스크립트가 추가할 정확한 줄을 출력했으니, 그걸 셸 설정 파일에 붙여넣으세요. |
| **특정 곡이 안 나온다** | 일부 곡은 로그인이 필요합니다. [쿠키로 로그인하기](#쿠키로-로그인하기-선택)를 보세요. |
| **어제까진 됐는데 이제 전부 재생이 안 된다** | 유튜브가 수시로 바뀌면서 yt-dlp를 망가뜨립니다. 업데이트하세요: `brew upgrade yt-dlp`(맥), `scoop update yt-dlp`(윈도우), 또는 패키지 매니저(리눅스). |
| **앨범 아트가 안 보인다** | 기본적으로 꺼져 있고 일부 터미널에서만 됩니다. 설정 → 일반에서 **앨범 아트**를 켜고 **재시작**하세요("재시작 후 적용" 설정입니다). |
| **키를 바꿨더니 모든 게 엉망이 됐다** | 설정 → 일반 → **단축키 초기화**. 다 용서됩니다. |

그래도 막혔다면, [GitHub 저장소](https://github.com/Ochichan/ytm-tui)에 이슈를 열고 무슨 일이 있었는지 적어주세요. 운영체제를 알려주면 가산점입니다.
